# HotStock

Put your money where your mouth is. A web app for running stock picking competitions among friends.

## Quick Start

```bash
bun install
bun run index.ts
# Visit http://localhost:3000
```

## Docker Deployment

```bash
docker compose up --build
# Visit http://localhost:3000
```

## Tech Stack

- **Runtime**: Bun
- **Backend**: Hono
- **Database**: SQLite
- **Frontend**: Vanilla HTML/CSS/JS
